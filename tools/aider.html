Aider

Code Freemium

AI pair programming in your terminal. Aider lets you pair program with LLMs to edit code in your local git repository. Works with Claude 3.7 Sonnet, DeepSeek, OpenAI models, and almost any LLM including local models. Features automatic git commits, voice input, and top SWE Bench performance.

Try Aider View Pricing

Key Features

Terminal-Based

Work directly in your terminal with full git integration and command-line workflows.

Codebase Mapping

Creates a comprehensive map of your entire codebase for better context in large projects.

Multi-Language Support

Works with Python, JavaScript, Rust, Ruby, Go, C++, PHP, HTML, CSS, and dozens more.

Auto Git Commits

Automatically commits changes with sensible commit messages for clean version control.

Voice Input

Request features, test cases, or bug fixes using your voice for hands-free coding.

Visual Context

Add images and web pages to chat for screenshots, reference docs, and more.

Multiple LLM Support

Works with Claude, DeepSeek, GPT-4, o1, and can connect to local models.

Top SWE Bench Score

One of the highest scores on SWE Bench, solving real GitHub issues from popular projects.
