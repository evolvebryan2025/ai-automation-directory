Important Information

Rate Limits: As of August 2025, Anthropic enforces weekly rate limits for heavy Claude Code users to manage costs. Teams running at scale should plan for limits and budget for additional capacity.

Usage Resets: Usage limits reset every 5 hours across all Claude platforms, not just Claude Code.

Minimum Requirements: Claude Code requires at least a Pro subscription ($20/month) or API credits. The free plan does not support Claude Code access.
